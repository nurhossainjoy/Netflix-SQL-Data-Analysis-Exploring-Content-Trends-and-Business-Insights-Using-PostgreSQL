# 🎬 Netflix SQL Data Analysis
![Netflix Logo] ( )

## 📖 Project Overview

This project explores the Netflix Movies and TV Shows dataset using **PostgreSQL** to answer real-world business questions through SQL. The analysis focuses on content distribution, ratings, genres, release trends, directors, actors, and keyword-based content classification.

The project demonstrates practical SQL skills commonly required for **Data Analyst**, **Business Analyst**, and **BI Analyst** roles.

---

## 🎯 Objectives

The primary objectives of this project are to:

- Analyze Netflix's content library using SQL.
- Solve real-world business questions through data analysis.
- Practice advanced SQL techniques for analytical reporting.
- Build a professional SQL portfolio project for GitHub.

## 📊 Business Questions Solved

This project answers several practical business questions, including:

1. Count Movies and TV Shows available on Netflix.
2. Find the most common content ratings.
3. List movies released in a specific year.
4. Identify the top content-producing countries.
5. Find the longest movie available.
6. Retrieve content added in the last five years.
7. Find movies directed by a specific director.
8. List TV Shows with more than a specified number of seasons.
9. Count content available in each genre.
10. Find movies featuring a particular actor.
11. Determine how many movies Salman Khan appeared in during the last ten years.
12. Categorize content as **Good** or **Bad** based on keywords in the description.
13. Analyze yearly Netflix content additions by country.
14. Calculate yearly percentage contribution of content.
15. Perform exploratory analysis using advanced SQL techniques.

## 📈 Key Insights

- Movies constitute the majority of Netflix's content library.
- The United States and India contribute the largest number of titles.
- Netflix experienced rapid growth in content additions after 2015.
- TV-MA is one of the most common maturity ratings.
- Content production varies significantly across countries and years.
- SQL can efficiently solve real-world analytical and reporting problems using a single dataset.
